Buat stiker WhatsApp pribadimu dengan mudah menggunakan gambar, foto, atau GIF.

### Cara Menggunakan

Ada dua metode yang bisa kamu gunakan:

<Steps>
  <Step title="Metode 1: Kirim Gambar">
    Kirim gambar secara langsung ke bot, lalu tulis caption:
    ```bash
    .sticker
    ```
  </Step>
  <Step title="Metode 2: Reply Gambar">
    Jika ada gambar yang sudah terkirim di chat, **Reply** (balas) gambar tersebut dan ketik:
    ```bash
    .sticker
    ```
  </Step>
</Steps>

### Opsi Tambahan

| Perintah | Fungsi |
| :--- | :--- |
| `.s` | Alias singkat untuk `.sticker` |
| `.sgif` | Untuk membuat stiker bergerak (dari video pendek/GIF) |
| `.smeme <atas>|<bawah>` | Membuat stiker dengan teks meme di atas/bawah |

<Note>
Untuk hasil stiker bergerak (`.sgif`), pastikan durasi video/GIF di bawah 10 detik agar proses konversi tidak gagal.
</Note>
