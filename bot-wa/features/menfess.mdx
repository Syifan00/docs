Fitur **Menfess** (Mention Confess) memungkinkan kamu mengirim pesan ke nomor WhatsApp lain melalui bot tanpa ketahuan identitas pengirimnya. Cocok untuk mengungkapkan perasaan atau sekadar mengirim pesan anonim.

### Format Perintah

Gunakan format berikut di chat pribadi (PC) ke bot:

```bash
.menfess <nomor_tujuan>|<pesan>
```

Atau jika ingin menyertakan nama samaran:

```bash
.menfess <nomor_tujuan>|<nama_samaran>|<pesan>
```

### Contoh Penggunaan

<CodeGroup>
```bash Tanpa Nama Samaran
.menfess 6281234567890|Hai, sebenernya aku ngefans sama kamu.
```

```bash Dengan Nama Samaran
.menfess 6281234567890|SecretAdmirer|Semangat ya kerjanya hari ini!
```
</CodeGroup>

<Warning>
**Penting:**
1. Nomor tujuan wajib menggunakan kode negara (contoh: 628xxx), bukan 08xxx.
2. Jangan mengirim pesan yang mengandung ancaman atau teror. Admin dapat melacak pengirim jika ada laporan penyalahgunaan.
</Warning>
```
