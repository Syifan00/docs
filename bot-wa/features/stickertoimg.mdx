Kebalikan dari fitur pembuat stiker, fitur ini berguna jika kamu melihat stiker bagus dan ingin menyimpannya ke galeri sebagai foto biasa.

### Cara Menggunakan

1. Pilih stiker yang ingin diubah.
2. **Reply** (balas) stiker tersebut.
3. Ketik perintah:

```bash
.toimg
```

Bot akan memproses dan mengirimkan file gambar (JPG/PNG) dari stiker tersebut.

### Keterbatasan

<Accordion title="Kenapa stiker gerak tidak bisa?" icon="circle-question">
Saat ini fitur `.toimg` bekerja maksimal pada stiker statis (diam). 

Untuk stiker bergerak, konversi ke video (MP4) atau GIF memerlukan proses server yang lebih berat dan mungkin belum tersedia di versi bot saat ini.
